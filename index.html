ASCII Generator

Status: Complete

TimeFrame: Fall 2022 (~2 weeks)

Languages Used: HTML, CSS, JavaScript

Tools: Visual Studio Code

This is an application that can take any image (given it has an image url), and turn it into ASCII art. I used an external API from filestack in order to do this. I've given a variety of options to the user for how their art should be displayed, and I save their generated pieces to local storage so that they can easily be brought back up later in the sidebar. This is something that I've found fun to experiment with.
